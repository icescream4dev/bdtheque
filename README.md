# API Moteur de Bédéthèque

Ce projet est une API REST développée en Node.js / Express permettant de récupérer des informations sur des bandes dessinées (BD) francophones à partir de leur numéro ISBN.

## Objectif

Agréger des sources de données gratuites (initialement Google Books API) pour proposer un point d'entrée unique et simple permettant de récupérer :
- Le titre
- Les auteurs
- L'éditeur
- L'image de couverture
- L'année de publication

## Prérequis

- Node.js (LTS recommandé)
